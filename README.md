# 哈尔滨师范大学录取预测原型

## 打开方式

本地预览地址：

```text
http://127.0.0.1:5500/
```

也可以直接双击 `index.html` 打开。

## 当前功能

- 只聚焦哈尔滨师范大学。
- 当前内置地区为黑龙江。
- 支持物理类、历史类。
- 输入高考分数和全省位次后，系统输出各专业的保、稳、冲、风险档位。
- 2026 预测分数和预测位次会用橙色高亮。

## 数据维护

专业数据在 `app.js` 的 `admissions` 数组里。

每个专业的数据格式：

```js
{
  province: "黑龙江",
  track: "物理类",
  major: "数学与应用数学（师范类）",
  scores: {
    2023: { score: 504, rank: 14656 },
    2024: { score: 560, rank: 17241 },
    2025: { score: 530, rank: 22442 },
  },
}
```

如果后续拿到学校招生网 Excel 或 CSV，可以把完整专业数据批量转换成这个结构。

## 预测口径

第一版采用位次优先：

```text
预测位次 = 2025 位次 x 0.5 + 2024 位次 x 0.3 + 2023 位次 x 0.2
```

再结合 2026 年本科控制线变化做轻量修正。后续可以加入招生计划人数、专业热度、地区报考人数变化等因素。
