# 哈尔滨师范大学录取预测原型

## 当前功能

- 支持普通类、艺术类、体育单招、数据说明四个入口。
- 省份下拉已扩展到全国主要招生省份。
- 黑龙江普通类物理类、历史类已录入专业分和位次，可输出保、稳、冲、风险档位。
- 其他省份已预留入口，但没有导入官方专业线前只显示待导入，不输出伪预测。
- 艺术类和体育单招已独立展示口径说明，不再混用普通高考位次算法。

## 本地预览

```text
http://127.0.0.1:5500/
```

也可以直接打开 `index.html` 预览静态页面。

## 数据维护

普通类专业数据位于 `app.js` 的 `ordinaryAdmissions` 数组。
每条数据包含省份、科类、专业、2023-2025 年最低分和最低位次。

完整版本建议从学校本科招生网、各省考试院投档表或官方 Excel/PDF 导入，按省份、科类、批次、专业四个维度维护。

## 预测口径

普通类第一版采用位次优先：

```text
预测位次 = 2025 位次 x 0.5 + 2024 位次 x 0.3 + 2023 位次 x 0.2
```

再结合 2026 年参考线做轻量修正。艺术类和体育单招需要单独录入专业统考、综合分、专项考试等规则后再开放预测。
