# 哈尔滨师范大学录取预测原型

## 当前功能

- 支持普通类、艺术类、体育单招、数据说明四个入口。
- 省份下拉会从已导入数据里自动补齐，目前覆盖 31 个省区市。
- 普通类已导入 2023-2025 年哈尔滨师范大学招生网历年分数线，包含本科批、提前批、专项计划、省属公费师范生等批次，可输出保、稳、冲、风险档位。
- 艺术类已导入 2023-2025 年公开分数记录，可查询最低分、平均分、最高分和批次。
- 体育类已导入官网公开分数记录；体育单招仍需专项考试、文化考试和招生简章规则，页面先展示历年分数。
- 没有位次的记录会按分数参考，不会硬造位次。

## 本地预览

```text
http://127.0.0.1:5500/
```

也可以直接打开 `index.html` 预览静态页面。

## 数据维护

录取数据位于 `admission-data.js`。
普通类数据写入 `window.HRBNU_ORDINARY_ADMISSIONS`，艺术类数据写入 `window.HRBNU_ART_ADMISSIONS`，体育类数据写入 `window.HRBNU_SPORT_ADMISSIONS`。

当前数据由 `tools/fetch_hrbnu_official.py` 从学校本科招生网历年分数查询接口生成。后续如果要继续提升准确度，可以补各省考试院投档位次和当年招生计划。

## 预测口径

普通类第一版采用位次优先：

```text
预测位次 = 2025 位次 x 0.5 + 2024 位次 x 0.3 + 2023 位次 x 0.2
```

再结合 2026 年参考线做轻量修正。艺术类和体育单招需要单独录入专业统考、综合分、专项考试等规则后再开放预测。
